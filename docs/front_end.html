The Invoca Front End Tech Radar is maintained by the UI Guild. Anyone can propose updates via pull request.

Frameworks

React

Component oriented development framework.

Titan

Component Library with build tools and guidelines for development. Based on React and Material UI.

Backdraft

Older UI framework using Backbone. Some of it has been partially migrated to React.

Languages

JavaScript

Used for all current front end development with React

CoffeeScript

Functional programming oriented language that transpiles to JavaScript. Used in legacy UI components.

TypeScript

Superset of JavaScript that provides compile time checking of code.

Libraries

jQuery

Necessary for Backbone but should not be integrated with React
  • Favor fetch over $.ajax
  • Use native DOM apis for HTML element state through refs in the rare case an escape hatch is necessary
  • React already manages the DOM, further manipulating it with jQuery can put you in an uncertain state.

Underscore

Necessary for Backbone but has been used outside that code.

Lodash

A more modern take to Underscore but almost completely overlaps with it. Since we canâ€™t remove underscore until Backbone is gone, we should not be using Lodash

Apollo

Data fetching library for GraphQL

D3

Visualization Library

Grunt

JS task runner. Overlaps with Webpack

Moment

Time and time zone library. No longer developed, less useful with modern browsers.

Testing

Jasmine

Test DSL used for our older UI tools.

Karma

Test runner for Jasmine.

Jest

Test framework for our React components.

Enzyme

Matcher library and renderer for current React tests.

React Testing Library

Candidate for replacing Enzyme with a more user centric testing approach.
